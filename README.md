# 📝 MiniBlog

Aplicação de blog desenvolvida com **React**, focada no consumo de **API REST** e integração com **Firebase**, incluindo autenticação de usuários e funcionalidades completas de interação.

> [!IMPORTANT]
> Projeto completo com autenticação, comentários, busca, filtros e sistema de categorias.

O projeto simula um ambiente real de blog, com gerenciamento de posts, sistema de comentários e experiência dinâmica em tempo real.

---

## 🚀 Tecnologias utilizadas

- ⚛️ React  
- 🔥 Firebase (Authentication + Firestore)  
- 🌐 API REST  
- 🎨 CSS3  
- 🧠 JavaScript (ES6+)  

---

## 💡 Funcionalidades

- 🔐 Autenticação de usuários (login e cadastro)  
- 📄 CRUD completo de posts  
- 📰 Listagem dinâmica de posts via API  
- 🔍 Visualização individual de posts  
- 💬 Sistema de comentários por post  
- 🔎 Busca e filtros de conteúdo  
- 🏷️ Sistema de categorias/tags  
- 🔄 Atualização em tempo real com Firebase  
- 📱 Interface responsiva e moderna  

---

## 🎯 Objetivo do projeto

Este projeto foi desenvolvido para:

- Consolidar conhecimentos em **React**  
- Praticar consumo de **API REST**  
- Implementar autenticação e banco de dados com **Firebase**  
- Simular uma aplicação real com múltiplas funcionalidades
